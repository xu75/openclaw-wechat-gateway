# 微信公众号文章写作规范

## 一、标题规范

### 基本原则
- **字数**：13-20字为宜，最长不超过32字
- **不用特殊符号**：标题不支持 emoji、全角符号等特殊字符
- **避免标题党**：标题要与内容相关，不要模棱两可

### 高点击标题公式
| 类型 | 示例 |
|------|------|
| **数字法** | 5个方法让你的效率翻倍 |
| **对比法** | 同样做这件事，为什么他成功了 |
| **悬念法** | 这个秘密，90%的人不知道 |
| **提问法** | 为什么 xxx 会 yyy？ |
| **热点法** | 结合时事/热点话题 |

---

## 二、排版规范

### 字体与字号
| 元素 | 建议 |
|------|------|
| **正文字号** | 14-16px（15px 最舒适） |
| **标题字号** | 18-22px |
| **注释/来源** | 12px |

### 间距设置
| 参数 | 建议值 |
|------|--------|
| **行间距** | 1.6-2.0（1.75 最佳） |
| **字间距** | 1-2px |
| **段落间距** | 空一行，15-20px |
| **页边距** | 左右各 10-15px |

### 配色原则
- **正文颜色**：#3F3F3F 或 #595959（不用纯黑 #000000）
- **强调色**：1-2种主色，不超过3种
- **避免**：高饱和度颜色、大面积亮色背景

---

## 三、文章结构

### 标准结构
```text
1. 开头（100-200字）
   - 引出话题/制造悬念/抛出观点

2. 正文（分段、加小标题）
   - 每段 3-5 行，不超过 6 行
   - 每 200-300 字加一个小标题
   - 关键信息用粗体/颜色标注

3. 结尾（100-150字）
   - 总结观点/引发思考/引导互动
```

### 小标题技巧
- 简短有力（5-10字）
- 可用数字序号（01、02、03 或 一、二、三）
- 统一格式风格

## 四、图片规范

| 参数 | 建议 |
|------|------|
| **封面图** | 900×383px（2.35:1 比例） |
| **正文图** | 宽度 1080px，高度不限 |
| **格式** | JPG/PNG，单张 < 5MB |
| **位置** | 每 300-500 字配一张图 |

### 配图要求
- 如果生成了文章的主题图片，**必须将图片内嵌到文章内容中**（通常放在标题下方、正文开头之前）

---

## 五、常见问题

### 标题
- ❌ 不要用 emoji 或特殊符号
- ❌ 不要超过 32 字
- ✅ 可以用数字增加吸引力

### 排版
- ❌ 不要用纯黑色文字
- ❌ 段落不要太长（手机阅读）
- ✅ 适当留白，增加可读性

### 图片
- ❌ 不要用模糊/水印图
- ❌ 不要图片尺寸不一致
- ✅ 封面图用 2.35:1 比例
